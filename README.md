### 핵심 기능

- REST API
  - 정류소 검색
  - 탑승 가능한 가장 빠른 버스 리스트 조회</br>

- 소켓 통신
  - 버스 기사에게 다음 정류소에서 승차, 하차 대기 중인 교통 약자 수 알림
  - 승객에게 탑승할 버스 도착 전 알림 
  - 승객에게 하차 역 도착 전 알림</br>
